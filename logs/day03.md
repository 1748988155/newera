# Arduino 学习日志 — Day 3 预备

## 今日学习内容

1. **按钮状态检测与边沿判断**

   * 用 `status != lastStatus` 判断按钮是否发生变化。
   * 使用 `INPUT_PULLUP`，按下为 `LOW`。
   * 记录 `lastStatus` 实现按下/松开检测。

2. **按压时间计算**

   * 用 `millis()` 记录按下时间 `pressStart`。
   * 按住按钮时按压时间实时计算 `millis() - pressStart`。
   * 区分松手统计和按住统计逻辑。

3. **LED 状态控制**

   * 使用 `ledStatus = !ledStatus` 翻转LED状态。
   * 使用非阻塞方法 (`millis() - blinkTime`) 控制闪烁。
   * 完成 LED 常亮、慢闪、快闪逻辑。

4. **长中短按模式判断**

   * 短按 `<500ms` → 常亮
   * 中按 `500–2000ms` → 慢闪
   * 长按 `>2000ms` → 快闪
   * 使用 `&&` 处理区间判断。

5. **嵌入式思维训练**

   * 使用状态机结构拆分按键逻辑与LED控制逻辑。
   * 理解模式选择和动作执行分离。
   * 分析程序执行时间线，理解变量更新时机。

## 今天的问题与解决

* **LED常亮问题**

  * 原因：`delay()`阻塞导致LOW状态太短。
  * 解决：使用非阻塞翻转控制 (`millis() - blinkTime`)。

* **按压时间只在松手后有效**

  * 原因：`openTime`只在松手更新。
  * 解决：按住时实时计算 `millis() - pressStart`。

* **区间判断错误**

  * 原因：`500 < pressTime < 2000` 在C++中不可用。
  * 解决：拆成 `pressTime > 500 && pressTime < 2000`。
